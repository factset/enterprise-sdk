# Period

several periods for the analysis can be specified - one for each period to be analyzed. When a period parameter is provided in the request, the API will 'cut' the returns for the period from the returns for the time window, specified in timeWindow parameter if provided, or from the lifetime returns, if no timeWindow parameter is provided
