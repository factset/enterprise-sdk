# Adjust

Controls the split and spinoff adjustments for the prices.   * **SPLIT** = Split ONLY Adjusted. This is used by default.   * **SPLIT_SPINOFF** = Splits & Spinoff Adjusted.   * **DIV_SPIN_SPLITS** = Dividend adjustments, Spinoff, and Splits combined.   * **UNSPLIT** = No Adjustments.
